# Design
This file documents the main philosophy behind the editor.

## Commandments
* Everything is a text buffer. As this is a text editor using
  text buffers as interfaces for other things like directorie
  just makes sense.
* Be practical this is not a resarch project into CRDT it is text
  editor made for me to be used for all my text editing tasks.
* Minimal config with sane defaults. Configuring things wastes users time.
